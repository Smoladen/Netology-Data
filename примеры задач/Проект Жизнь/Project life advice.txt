check_if_alive - так делать можно. Чтобы было компактнее можете проходить двойным циклом. Псевдокод следующий:

обнуляем счетчик живых
Пробегаем в окрестностях от -1 до +1 относительно клетки по x    
     Пробегаем в окрестностях от -1 до +1 относительно клетки по y    
     Пропускаем края поля вселенной и центр        
     Если клетка живая, то увеличиваем счетчик
возвращаем счетчик живых
Код примерно в таком духе:

/*     -1 0 1  
     -1 - - -  
      0 - * - 
      1 - - - 
*/   
//pI,pJ - позиция текущей клетки, для которой смотрим соседей   
//world - входной массив   
//rows, cols - количество строк и столбцов в массиве  
int lifeCount = 0;       
for( int i = pI - 1; i <= pI + 1; i++ ){         
	if( i < 0 || i >= rows) {           
		continue;         
	}         
	for( int j = pJ - 1; j <= pJ + 1; j++ ){       
		if( i < 0 || j >= cols){             
			continue;           
		}           
		if( i == i && j == j ){             
			continue;           
		}            
		if( world[i][j] == 1 ){            
			lifeCount++;                        
		}
	}         
}
