
Чтобы создать массив указателей на объекты MyClass, каждый из которых должен быть инициализирован параметром при создании, вы можете воспользоваться следующим кодом:

#include <iostream>

class MyClass {
public:
    // Конструктор класса, принимающий параметр
    MyClass(int param) : param(param) {}

    // Пример метода, использующего параметр
    void printParam() {
        std::cout << "Param: " << param << std::endl;
    }

private:
    int param;
};

int main() {
    const int size = 10;
    MyClass* pArray[size];

    // Создание массива объектов и их инициализация параметрами
    for (int i = 0; i < size; ++i) {
        pArray[i] = new MyClass(i); // Передача параметра при создании
    }

    // Пример использования объектов массива
    for (int i = 0; i < size; ++i) {
        pArray[i]->printParam(); // Вызов метода объекта
    }

    // Очистка памяти, выделенной под объекты
    for (int i = 0; i < size; ++i) {
        delete pArray[i];
    }

    return 0;
}


Этот код создает массив указателей pArray, содержащий 10 указателей на объекты MyClass. При создании каждого объекта вызывается конструктор MyClass, которому передается параметр i. В последующем каждый объект из массива может быть использован как обычный объект класса MyClass. Не забудьте освободить память с помощью delete, чтобы избежать утечек памяти.
